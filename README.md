# 🧩 Custom 404 Error Page UI

A **modern 404 error page design** created in **Figma**.

Instead of a boring error screen, this design introduces a friendly character and playful layout to improve the user experience.

## ✨ Features

* Friendly mascot illustration
* Large background error number
* Minimal UI layout
* Clear call-to-action button
* Soft pastel color palette

---

## 🎨 Design Style

* Playful UI
* Clean and minimal layout
* Friendly user interaction design

---

## 🛠 Tools Used

* **Figma**
* UI/UX design principles
* Typography and spacing
* Minimal interface design

---

## 📌 Purpose

This design demonstrates how error pages can be turned into **engaging user experiences** rather than frustrating moments.
